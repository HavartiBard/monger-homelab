Manual install process:
```curl -sfL https://get.k3s.io | sh -```

verify install:
```systemctl status k3s```

Check default kube objects:
```sudo kubectl get all -n kube-system```

Check kubeconfig:
```sudo cat /etc/rancher/k3s/k3s.yaml```
